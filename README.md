# packages-by

> Total published packages by an npm user.

## Install 

```
	$sudo npm install --save packages-by
```

## Usage

```js
	const packagesBy('rishi').then(user => {
		console.log(user);
		/*
		name: 'rishi',
		totalPackages: 10
		*/
	});
```
